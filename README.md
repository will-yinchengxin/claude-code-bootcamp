**7 天掌握 Claude Code 核心功能，从环境安装到多智能体编排**

## 项目简介

Claude Code 训练营是一个系统化的学习路径，专为希望深度掌握 Claude Code CLI 的开发者设计。无论你是初次接触 AI 辅助编程，还是希望将 Claude Code 集成到你的开发工作流中，本训练营都将帮助你从零基础成长为能够构建 AI 辅助开发平台的工程师。

## 学习路径

| 天数 | 主题 | 核心技能 |
|------|------|----------|
| Day 1 | 环境安装 + CLAUDE.md + Prompt 工程基础 | 环境配置、项目上下文管理 |
| Day 2 | Slash Commands + Skills | 复用工作流、效率提升 |
| Day 3 | MCP 原理 + 配置现有 MCP Server | 外部工具集成、数据连接 |
| Day 4 | 自己动手写一个 MCP Server | 自定义工具开发 |
| Day 5 | Hooks：自动化与工作流守卫 | 生命周期拦截、自动化 |
| Day 6 | Sub-Agents + Agent SDK | 多智能体编排 |
| Day 7 | 综合实战：从零构建 AI 辅助后端工程平台 | 端到端集成开发 |

## 为什么这么做

- **循序渐进**：每天一个核心概念，稳步提升技能水平
- **实战导向**：每个知识点都配有动手实验和代码示例
- **完整项目**：第七天带你构建完整的 AI 辅助后端平台
- **中文教程**：全中文内容，术语准确，讲解清晰

## 快速开始

```bash
# 克隆项目
git clone https://github.com/will-yinchengxin/claude-code-bootcamp.git

# 开始 Day 1
cd curriculum/day-01-basics
```

## 内容大纲

### Day 1：基础入门

学习内容：
- Claude Code CLI 安装与认证
- CLAUDE.md 文件的作用与语法
- Prompt 工程基础：如何与 AI 有效沟通

### Day 2：命令与技能

学习内容：
- Slash Commands 详解
- Skills 工作流复用
- 自定义命令配置

### Day 3：MCP 集成

学习内容：
- Model Context Protocol 原理
- 配置现有 MCP Server
- 连接数据库、文件系统等外部资源

### Day 4：自定义 MCP Server

学习内容：
- 使用 Python 开发 MCP Server
- 定义 Tools、Resources、Prompts
- 注册并使用自定义工具

### Day 5：Hooks 自动化

学习内容：
- 生命周期钩子原理
- pre-response、post-response 拦截
- 工作流守卫与自动化

### Day 6：智能体编排

学习内容：
- Sub-Agents 子智能体
- Agent SDK Python 开发
- 多智能体协作模式

### Day 7：综合实战

学习内容：
- 从零构建后端工程
- 集成所有学到的技能
- 构建 AI 辅助开发平台
